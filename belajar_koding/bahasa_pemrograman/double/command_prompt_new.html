Mengelola File Dan Folder

Daftar Isi
  • Definisi
  • Kode
Definisi

Kode

  • dir : berfungsi untuk menampilkan daftar isi folder saat ini.

    ls

  • cd

    • Berfungsi untuk masuk kedalam folder. Ketik "cd" diikuti dengan nama folder yang dituju.

      cd documents

    • Berfungsi untuk keluar dari folder.

      cd ..

  • md : Berfungsi untuk membuat folder baru. Ketik "md" diikuti dengan nama folder baru yang ingin dibuat.

    md NewFolder

  • rd

    • Berfungsi untuk menghapus folder kosong. Ketik "rd" diikuti dengan nama folder baru yang ingin dihapus.

      rd NewFolder

    • Berfungsi untuk menghapus folder yang ada isinya. Kode "/s" untuk menghapus semua isi folder, "/q" untuk menghapus tanpa konfirmasi

      rd /s /q NewFolder

  • mv

    • Berfungsi untuk mengubah nama folder. Ketik "mv" diikuti dengan nama folder lama, dan diikuti dengan nama folder baru.

      mv docs documents

    • Berfungsi untuk mengubah nama file. Ketik "mv" diikuti dengan nama file lama, dan diikuti dengan nama file baru.

      mv FileLama.txt FileBaru.txt

    • Berfungsi untuk memindah file atau folder. Ketik `mv` diikuti dengan nama file atau folder yang ingin dipindahkan, lalu diikuti dengan lokasi tujuan.

      mv NewFile.txt /home/user/Documents/

  • touch : Berfungsi untuk membuat file baru. Ketik "touch" diikuti dengan nama file yang ingin dibuat.

    touch NewFile.txt

  • rm : Berfungsi untuk menghapus file. Ketik "rm" diikuti dengan nama file yang ingin dihapus.

    rm NewFile.txt

  • echo

    • Berfungsi untuk menampilkan teks.

      echo "Hello World"

    • Berfungsi untuk menulis isi file dengan menimpa teks yang sudah ada.

      echo "Hello World" > NewFile.txt

    • Berfungsi untuk menulis isi file dan menambah teks yang sudah ada.

      echo "Hello World" >> NewFile.txt

  • cat : Berfungsi untuk menampilkan isi file. Ketik "cat" diikuti dengan nama file yang ingin ditampilkan.

    cat NewFile.txt

  • cp : Berfungsi untuk menyalin file atau folder. Ketik `cp` diikuti dengan nama file atau folder yang ingin disalin, lalu diikuti dengan lokasi tujuan.

    cp NewFile.txt /home/user/Documents/
